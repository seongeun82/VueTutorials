# 환경구성

- 기본적으로 npm 이나 yarn 이 설치되어 있어야 한다.

```
sudo yarn add global @vue/cli
vue --version
```

### 프로젝트 생성

```
vue create 프로젝트명\
```

### 프로젝트 실행

```
yarn serve
```

- 실행화면
